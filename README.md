# 🚗 BMW Hub – The Ultimate Driving Experience

An interactive, responsive, and modern landing page showcasing iconic BMW cars, built entirely with **HTML, CSS, and basic JavaScript**. Created as a **Semester 1 Web Development Project**.

---

## 🌟 Key Features

*   **🎨 Dynamic Theme Customizer**: Change the entire website's accent colors (M Power Blue, Sport Red, M Sunset Gold) instantly with customizer dots in the hero section.
*   **🔎 Interactive Specs Modal (Lightbox)**: Click on any BMW model card to view key performance specifications (Horsepower, 0–100 km/h times, Top Speed, Est. Price) in a premium detailed modal.
*   **✉️ Instant Toast Notifications**: Fully simulated test-drive booking system with clean, animated toast alerts notifying successful submissions.
*   **🌊 Frosted Glass Sticky Navbar**: Interactive header that shrinks and becomes blur-translucent on scroll.
*   **📱 Responsive Layout**: Fully optimized for Desktop, Tablets, and Mobile screens.

---

## 🛠️ Tech Stack Used

*   **HTML5**: Semantic elements, forms, and custom data attributes.
*   **CSS3**: Custom properties (CSS variables) for themes, grid and flexbox layouts, hover transitions, and animations.
*   **JavaScript (ES6)**: DOM manipulation, event listeners, dynamic modals, classes toggles, and notification alerts.
*   **FontAwesome (v6)**: Vector icons for tags and modal specifications.
